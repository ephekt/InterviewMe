Find two items in a list quickly...

Given a list of a million signed integers we want to find two that add up to zero. Space complexity is not important initially, however the speed to which we find these numbers is. The integers are stored in an unsorted array. The number of items in the array is not that important. It could be 1 billion tomorrow.

View Our Answer

Use a hash. Store each vale you have seen in the hash. As you iterate over new items, negate the value and see if the opposite value is already in the hash. If so, you're done. If not, keep looking.
